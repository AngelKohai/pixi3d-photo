# Pixi3D - Outil photo du mois V2

Outil web pour créer une planche 10 x 15 cm avec des photos 35 x 45 mm.

## Fonctionnement

L'utilisateur peut importer de 1 à 6 photos.

- 1 photo : 6 copies
- 2 photos : 3 copies chacune
- 3 photos : 2 copies chacune
- 4 à 6 photos : 1 copie chacune

Les photos sont recadrées individuellement au format 35 x 45 mm.

## Fichiers

- `index.html`
- `style.css`
- `app.js`

## Mise à jour sur GitHub

1. Ouvrez le dépôt `pixi3d-photo`.
2. Remplacez les anciens fichiers par ceux-ci.
3. Cliquez sur `Commit changes`.
4. GitHub Pages mettra le site à jour automatiquement.
