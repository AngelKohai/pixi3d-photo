# Pixi3D - Outil photo du mois

Petit outil web pour recadrer une photo au format 35 x 45 mm et générer une planche 10 x 15 cm.

## Fichiers

- `index.html`
- `style.css`
- `app.js`

## Mise en ligne avec GitHub Pages

1. Créez un dépôt GitHub, par exemple `pixi3d-photo`.
2. Ajoutez ces fichiers dans le dépôt.
3. Allez dans **Settings > Pages**.
4. Dans **Build and deployment**, choisissez :
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. GitHub vous donnera une adresse du type :
   `https://votre-compte.github.io/pixi3d-photo/`

## Domaine personnalisé

Vous pourrez ensuite connecter un sous-domaine comme :

`photo.pixi3d.fr`

dans les réglages GitHub Pages.
